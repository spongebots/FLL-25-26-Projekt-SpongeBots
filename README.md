# FLL 25/26 Projekt SpongeBots
Als Robotikteam SpongeBots haben wir für die FLL Saison 25/26 unter dem Thema “Wie kann ich einem Archäologe bei der Arbeit behilflich sein?” folgendes Projekt entwickelt, um die Funddokumentation zu verbessern.

Unser Projekt besteht aus einem [längenverstellbaren Maßstab](massstab/README.md) und einer [digitalen Fototafel](fototafel/README.md).
